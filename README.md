# **Insect Detection Using Deep Learning Network**

### In agriculture, insect pests are always one of the main factors affecting product quality. Accurate detecting insect pests can help famers to prevent it's effects and improve their product. In this project, we propose a novel method that enables the automatic detection of insect pests. We use the *[IP102-Dataset](https://www.kaggle.com/datasets/rtlmhjbn/ip02-dataset)* on Kanggle to train out model.
